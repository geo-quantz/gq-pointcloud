# GeoQuantz PointCloud Pipeline

国土地理院マニュアルに準拠した地上レーザー(TLS)およびUAVレーザーの点群フィルタリングツールです。

## 主な機能
- **プリセット選択**: `--preset tls` (15m制限/10mmボクセル) または `--preset uav` を選択可能。
- **真の器械点抽出**: E57ヘッダーから自動で設置位置を取得。
- **動体除去**: オルソ画像を参照し、車両等のゴーストを自動排除。

## ディレクトリ構成
- `01_raw/`: 入力E57を配置
- `02_ref/ortho/`: 参照オルソ画像を配置
- `04_product/`: 成果品(LAS)とレポートが出力される
